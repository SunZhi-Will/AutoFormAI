如何使用 AutoFormAI

AutoFormAI 是一個使用 Google Gemini AI 模型自動填寫 Google Form 的工具，讓您不再需要手動填寫重複性的表單。

提示： 使用前請確保您已經有一個 Google 帳戶，並且有權限創建 API 金鑰。

獲取 Gemini API 金鑰

1

訪問 Google AI Studio 並登入您的 Google 帳戶

2

點擊「Create API key」按鈕創建新的 API 金鑰

3

複製生成的 API 金鑰（它看起來像這樣：AIzaSyC_...）

4

在 AutoFormAI 應用中填入您的 API 金鑰

指定 Google Form

1

直接粘貼 Google Form 的網址到輸入框中

2

系統會自動嘗試檢測表單的結構，並在檢測成功後啟用「生成填寫連結」按鈕

3

如果檢測失敗，請確認您的網址是否正確，或嘗試使用其他瀏覽器

提供您的資訊

1

在文本框中輸入您希望 AI 用來填寫表單的資訊，格式不限。例如：

名字: Sun
性別: 男
電話: 0912345678
出生:{"year": 1999,"month": 8,"day":19}
Email: xxxx@gmail.com
地址: 高雄市
喜好: 畫畫、遊戲
2

您可以使用自然語言或結構化格式，AI 會盡可能理解您的意圖

提示：結構化格式（例如 JSON）通常能提高 AI 理解準確性

生成填寫連結

1

確認所有資訊無誤後，點擊「生成填寫連結」按鈕

2

系統會顯示以下內容：

  • 表單結構 - 顯示 AI 將要回答的問題
  • 填寫連結 - 點擊此連結即可打開已預填答案的表單
3

點擊「複製連結」按鈕可以方便地複製填寫連結

常見問題

我的 API 金鑰安全嗎？

您的 API 金鑰僅用於從瀏覽器直接向 Google API 發送請求，不會被發送到任何其他服務器或儲存。所有處理都在您的本地瀏覽器中完成。

使用 AutoFormAI 需要付費嗎？

本工具完全免費使用。唯一可能的費用來自 Google Gemini API 的使用（如果您超過了免費額度）。大多數用戶都不會超過 Google 提供的免費額度。

支持哪些類型的表單？

目前僅支持 Google Form。我們正在努力未來支持更多平台，如 Microsoft Forms、TypeForm 等。

回答會永遠正確嗎？

雖然 AI 會盡力提供合適的答案，但準確性可能因表單類型和您提供的資訊而異。始終建議在提交前檢查自動填寫的答案，特別是對於重要的表單。

我可以保存表單設置嗎？

目前不支持保存設置，每次使用都需要重新輸入資訊。我們計劃在未來版本中增加設置保存功能。
